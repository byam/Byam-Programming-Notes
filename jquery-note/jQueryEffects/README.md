## jQuery Effects

### jQuery UI

```js
// include jQuery UI library
<script src="//ajax.googleapis.com/ajax/libs/jqueryui/1.9.1/jquery-ui.min.js"></script>



// explode
$('div').effect('explode');

// bounce
$('div').effect('bounce', {times:2}, 200);

// draggable
$('#car').draggable();

// resize
$('div').resizable();

$('ol').selectable();

$('ol').sortable();

$('#menu').accordion();
```
